## Document Non-Claims

This matrix does not certify that any layer is compliant, does not authorize any layer to act, and does not constitute a governance decision. It is a reference document for reviewer-side verification of explicit non-claims. Citing this matrix does not satisfy a layer's obligation to state its own non-claims.

---

## Anchor Invariant

**Non-claims do not propagate by reference.** A layer that cites another layer's non-claims does not thereby inherit those non-claims for its own artifact. Each primitive must state its own negative surface explicitly.
