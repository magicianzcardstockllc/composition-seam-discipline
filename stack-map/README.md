**Authority:** James Aull / MagicianzCardstock LLC

---

## Purpose

This folder contains the Living Governance Stack Map — a public map of the modular AI governance problem space organized by layer position.

The map identifies the unsolved question each layer addresses, the specific gap that remains in current practice, and where the Seam Discipline enables comparison between layers without collapsing their boundaries.

The map is a topology of the problem, not a solution. It is not a standards proposal, a coalition announcement, or a claim that any listed layer solves the complete governance problem.
